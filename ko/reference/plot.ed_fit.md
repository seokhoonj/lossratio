# Plot an ED fit

Visualise an object of class `"ed_fit"` by delegating to
[`plot.ed()`](https://seokhoonj.github.io/lossratio/ko/reference/plot.ed.md)
on the underlying `ed` data stored in `x$ed`.

## Usage

``` r
# S3 method for class 'ed_fit'
plot(x, ...)
```

## Arguments

- x:

  An object of class `"ed_fit"`.

- ...:

  Arguments passed to
  [`plot.ed()`](https://seokhoonj.github.io/lossratio/ko/reference/plot.ed.md).

## Value

A `ggplot` object.
