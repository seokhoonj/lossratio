# Whether all of `cols` are present in `names(df)`

Whether all of `cols` are present in `names(df)`

## Usage

``` r
.has_cols(df, cols)
```

## Arguments

- df:

  A data.frame (or similar with
  [`names()`](https://rdrr.io/r/base/names.html)).

- cols:

  Character vector of column names.

## Value

Logical scalar.
