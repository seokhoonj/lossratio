# Plot an ata fit

Visualise an object of class `"ata_fit"` by delegating to
[`plot.ata()`](https://seokhoonj.github.io/lossratio/reference/plot.ata.md)
on the underlying `ata` data stored in `x$ata`.

## Usage

``` r
# S3 method for class 'ata_fit'
plot(x, ...)
```

## Arguments

- x:

  An object of class `"ata_fit"`.

- ...:

  Arguments passed to
  [`plot.ata()`](https://seokhoonj.github.io/lossratio/reference/plot.ata.md).

## Value

A `ggplot` object.
